## 🎬 演示场景 1: 职场效率自动化

**时长**: 3 分钟
**目标**: 展示 FeishuMind 如何提升职场效率

### 脚本

**场景设定**: 假设用户是一个忙碌的程序员

**步骤 1: 创建事件提醒** (1分钟)

```
👤 用户: "提醒我明天下午2点参加产品评审会"

🤖 FeishuMind:
"✅ 已为你创建日程提醒:
  时间: 明天 14:00-15:00
  主题: 产品评审会
  提醒: 提前15分钟、1小时、1天"
```

**演示要点**:
- 自然语言理解
- 自动创建飞书日历事件
- 多级提醒设置

**步骤 2: 快速查询** (1分钟)

```
👤 用户: "我明天的安排是什么？"

🤖 FeishuMind:
"你明天有以下安排:
  📅 09:00-10:00 团队站会
  📅 14:00-15:00 产品评审会
  📅 16:30-17:00 代码评审"
```

**演示要点**:
- 跨系统查询 (飞书日历)
- 结构化输出

**步骤 3: 智能建议** (1分钟)

```
👤 用户: "我今天有点累，想调整一下工作"

🤖 FeishuMind:
"理解你的感受。根据你今天的日程，我建议:
  1. 下午3点后无会议，适合专注工作

```

**演示要点**:
- 情绪识别
- 个性化建议

```

```

   • 用户数据完全掌控
